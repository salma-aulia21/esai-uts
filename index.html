Rangkuman Kegiatan Praktikum SIGWEB Minggu 1-7

Pada pembelajaran di Kuliah Praktikum SIG WEB, saya banyak mempelajari mengenai penggunaan HTML, CSS, JavaScript, Bootstrap, LeafletJS, serta penggunaan Geojsn. Dari minggu pertama hingga minggu kedua mata kuliah ini dilaksanakan, saya masih mempelajari dasar-dasar dari Mata Kuliah Praktikum SIG WEB ini. Dimulai dengan mengenali apa itu web, webpage, serta pengenalan struktur dasar HTML, yang kemudian ditutup dengan tugas individu untuk membuat sebuah tampilan hasil seperti pada contoh dengan menggunakan HTML.

Pada minggu ketiga dan keempat, kurang lebih memiliki metode pembelajaran dan materi yang sama. Namun, tugas pada minggu keempat ini memiliki tantangan lebih, yaitu saya diinstruksikan untuk mencontoh hasil keluaran yang sudah ada, tetapi kini hasil keluaran tersebut memiliki tampilan yang lebih baik, yaitu dengan menggunakan CSS di dalam HTML tersebut.

Pada minggu kelima, saya diberi arahan unuk memodifikasi sebuah template CV yang tersedia di internet dengan menggunakan sumber HTML dan CSS dari yang asli. Menurut saya, praktikum minggu kelima ini sangat seru karena bisa mengembangkan kreatifitas saya.

Pada minggu keenam, saya mendapatkan materi baru, yaitu mengenai Leaflet. Saya menyadari bahwa sering kali saya banyak menggunakan web yang menggunakan leaflet. Maka dari itu. Pada praktikum minggu keenam ini, saya diinstruksikan untuk membuat sebuah halaman web yang menampilkan penggunaan leaflet, yang kemudian saya membuat sebuah halaman web yang menampilkan Peta Landmark di Indonesia.

Pada minggu ketujuh, masih dengan materi leaflet. Namun, pada kali ini, pada file HTML harus disertakan masukan file Geojsn. Saya memilih untuk membuat peta batas administrasi yang menampilkan batas-batas kelurahan di Kota Semarang dan memiliki point marker dan pop up nya untuk menampilkan nama kecamatan tersebut. Pada praktikum kali ini juga saya harus menggunakan perangkat lunak QGIS untuk bisa menghasilkan file js.
